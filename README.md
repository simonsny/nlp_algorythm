# nlp_algorythm
NLP to make summaries of text documents.


Creaste conda env

    - `conda create -n nlp_becode python=3.6`
 
 
 Activate env:
 
     - `conda activate nlp_becode`


Install flair:

     - `pip install flair`
 
 
Installing spacy

    - `conda install -c conda-forge spacy`

    - `python -m spacy download en_core_web_sm`
